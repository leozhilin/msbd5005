TASK B / Media Bias and Entity Framing Analysis

This page is designed to identify potential bias in how news sources represent people, places, organizations, and events. The analysis has three layers: Attention Bias: which actors each outlet selectively emphasizes; Framing Bias: how each outlet frames key entities relative to the overall average; Event Bias: how each outlet explains events such as kidnapping, protests, environmental damage, and police investigations relative to the overall average. Click a heatmap cell to view the corresponding original evidence snippets. For entity cells, the linked word cloud will also update to show outlet-specific contextual words.

Analysis Modes

The key purpose of Task B is not simply to count word frequency, but to compare how different media outlets selectively report specific people, places, and events. This page supports six visual components for the final analysis: the attention heatmap, the relative framing heatmap, the relative event narrative matrix, the entity context word cloud, the evidence snippets, and the overall interpretation.

1. Attention Bias

Shows which actors receive more coverage from different outlets. It helps identify whether some outlets overemphasize POK, GAStech, Government, or specific people.

2. Framing Bias

Shows whether an outlet describes POK, GAStech, or Government more negatively or less negatively than the overall average.

3. Event Bias

Shows how different outlets narrate the same events. It helps compare narrative differences in kidnapping, protests, environmental disputes, and police investigations.
