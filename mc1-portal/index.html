任务背景 (OPERATIONAL CONTEXT)

虚构国家 Kronos，天然气公司 GAStech 在 IPO 前后发生多名员工失踪；民间组织 保护克洛诺斯 (POK) 被列为嫌疑对象。本系统汇总官方提供的历史新闻、员工简历、两周内部邮件头，供分析「人—组织」关系前先行查阅原始材料。

注：本场景及人物、地点均为虚构。请在本地以 HTTP 方式打开本站点（见页脚），以便拉取 data 分片与 Word/文本原件。

统合档案室 (RAW MATERIALS)

在单页中浏览人事与新闻：点选人名可内嵌预览其简历/传略（Word 转 HTML），同页下方列出 email headers.csv 中匹配到的往来邮件头；在「新闻分馆」中选媒体、再点文章即加载 .txt 全文。统计（简历 — 份 · 新闻 — 篇）见下方入口。

进入统合档案室 人事 / 新闻 / 內郵 同一视窗，支持地址栏 # 分享

子任务分析入口

三页均基于已抽取的 *.json 与全文统计，图表与源数据保持可追溯对应。

任务 A 媒体目录 · 同题共现网 · 报道量 任务 B 实体归一化热力图 · 报道词云 任务 C 多源关系边 · 邮件通信子网
